# 🎬 Automated Content Creator Hub (ACCH)

> A comprehensive, cross-platform ecosystem designed to streamline video content management and automated processing for digital creators.

## 🚀 Overview

The **Automated Content Creator Hub (ACCH)** is built with a decoupled Hybrid-Cloud architecture. It empowers creators to upload raw media directly to the cloud and automatically processes the content in the background, offering a seamless experience across both Web and Mobile platforms.

By utilizing Presigned URLs and direct-to-cloud capabilities, the system completely bypasses the server for heavy file uploads, resulting in zero server bandwidth bottlenecks and infinite scalability.

## 🏗️ Monorepo Architecture & Code Quality

This project is structured as a Monorepo, containing three independent workspaces. The core architecture strictly adheres to **DRY, KISS, and YAGNI** principles, ensuring high maintainability.

* **`/backend`**: The core API and data processing factory (Node.js, Express, MongoDB). *Features centralized error handling, modular services, and background jobs.*
* **`/web`**: The Pro Workspace tailored for desktop users (React / Vite + Bootstrap) - *[In Progress]*
* **`/mobile`**: The "On-the-go" application for immediate mobile uploads (React Native) - *[Planned]*

## 💻 Tech Stack

### Control Plane & Processing (Backend)
* **Runtime:** Node.js / Express.js
* **Database:** MongoDB Atlas (Robust Schema Validation)
* **Storage Infrastructure:** Cloudflare R2 (S3-Compatible Object Storage)
* **Media Processing:** FFmpeg (via `fluent-ffmpeg` & `ffmpeg-static`)
* **Cloud SDK:** AWS SDK v3

### Data Plane (Frontend)
* **Web Client:** React / Vite / Bootstrap
* **Mobile App:** React Native / Expo *(Upcoming)*

## ✨ Key Features

* [x] **Direct-to-Cloud Uploads:** Secure, temporary Presigned URLs generated by the backend, allowing clients to upload massive video files directly to Cloudflare R2.
* [x] **Zero Memory Leak Storage:** Implemented Node.js `stream.pipeline` for safe buffer handling during media processing.
* [x] **Automated Video Transcoding:** Background jobs to compress raw videos into web-optimized H.264 formats.
* [x] **Smart Thumbnail Extraction:** Automated screenshot generation from video frames using FFmpeg.
* [x] **Shorts/Teaser Generation:** Auto-clipping the first 15 seconds of raw footage for preview purposes.
* [x] **Clean Architecture:** Refactored Controller-Service layers with centralized error handling and dynamic file routing.
* [ ] **Cross-Platform Access:** Unified dashboard across Web and Mobile.
